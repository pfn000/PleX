# Overview

PlexCode documentation now uses the root `/.docs` folder as the official website source.

## What this site does

- uses `Docs.PleX.html` as the visual shell
- auto-discovers markdown files in `/.docs`
- loads each markdown file as a website section without leaving the site
- publishes the root `/.docs` folder through GitHub Pages using a custom workflow
