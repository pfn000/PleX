# PlexCode documentation

This site is driven by the real markdown files committed in the repository's `docs/` folder and published through GitHub Pages.

## What this docs site covers

- how the GitHub Pages docs site is configured from `main` + `/docs`
- where the markdown source files live and how the static shell loads them
- how Plex-related file types are being prepared for a future upstream GitHub Linguist contribution

## Recommended reading order

1. Start with [Getting Started](./getting-started.md) to understand the docs workflow.
2. Open [Project Documentation Site](./README.md) for GitHub Pages setup details.
3. Review [Contributing PleX file types to GitHub Linguist](./github-linguist-upstream-custom-language.md) for the upstream language-definition plan.
