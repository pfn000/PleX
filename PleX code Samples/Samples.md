# Hello World
<span style="color:Green">*Correct*
```PleXCode
Show | Message
╰──➤ Hello World!
```

<span style="color:Red">*Incorrect*
```PleXCode
Argu~!! | Greeting
╰──➤ message | "Hello from PleX!"
```
> [!NOTE] 
> Why is this? 
> This is because `Show` is still an argument or "command" where it was originally seens as `show~!!` because 
> it is an ACTION. This is why using `Argu~!!` is silly becuase normally `Argu~!!` are used as Logic Commands or Mechanical.
